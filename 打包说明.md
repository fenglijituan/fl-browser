# FL Browser 打包指南

## 打包前准备

1. 安装依赖：
```bash
npm install
```

2. 优化文件大小（可选）：
```bash
node compress.js
```

## 打包命令

### Windows 便携版（推荐）
```bash
npm run build:zip
```
输出：`dist/FL Browser-1.0.0-win.zip`

### Windows 安装包
```bash
npm run build:installer
```
输出：`dist/FL-Browser-Setup-1.0.0.exe`

### 一键打包
双击运行：`一键打包.bat`

## 文件说明

- `index.html.bak` - 优化前备份
- `dist/` - 打包输出目录
- `updater.js` - 自动升级模块
- `compress.js` - 文件压缩工具

## 自动升级

浏览器启动时会自动检查更新，如果有新版本会提示用户下载。
升级数据保存在：`%APPDATA%/fl-browser/`
